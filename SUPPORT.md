# Support

## Getting Help

Use the following channels based on the kind of help you need:

- usage question or setup help: open a GitHub issue with the support or question template
- bug report: open a GitHub issue with the bug template
- feature request: open a GitHub issue with the feature template
- security issue: follow [SECURITY.md](/Users/guokeyu/AI/codex/token-usage-universal/SECURITY.md) and do not file a public issue

## Before Opening an Issue

Please include:

- your OS and Python version
- the command you ran
- whether you used local defaults or env overrides
- relevant `health`, `sources`, or `diagnose` output

## Scope

This repository is focused on local-first token accounting from local logs and local app data. General cloud billing disputes or upstream provider outages are usually out of scope.
